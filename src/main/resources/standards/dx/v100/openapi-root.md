# DCSA Document Exchange (DX) API

This is the OpenAPI specification of the **DCSA Document Exchange (DX)** cross-standard module.

This API allows the transfer of structured documents and asynchronous feedback between a DX Producer and a DX Consumer.

The registration of DX Consumers with DX Producers is out of scope.

The authentication and authorization in both directions between DX Producers and DX Consumers is out of scope.

### Work in progress ⚠️

The DCSA Document Exchange (DX) cross-standard module, including this API, is currently being designed and is **not** ready for general adoption yet.
